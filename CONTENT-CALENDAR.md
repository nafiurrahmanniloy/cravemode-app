## Content Quality Standards

### Every Blog Post Must Have:
- [ ] Keyword-optimized H1 (primary keyword within first 3 words when possible)
- [ ] Meta title (50-60 chars) + meta description (150-160 chars)
- [ ] Table of contents (for posts > 1,500 words)
- [ ] At least 1 original image or before/after comparison
- [ ] Internal links to 3+ related pages
- [ ] CTA mid-article + end-of-article
- [ ] FAQ section (3-5 questions) with FAQPage schema
- [ ] Author bio with credentials
- [ ] Published date + last updated date
- [ ] Schema: Article + Person

### Every City Page Must Have:
- [ ] City name in H1, title, meta description
- [ ] Local restaurant statistics
- [ ] Traditional photographer pricing for that city
- [ ] 3+ before/after examples
- [ ] Local testimonial if available
- [ ] Links to neighboring city pages
- [ ] Schema: Service + areaServed

### Every Service Page Must Have:
- [ ] Clear value proposition in H1
- [ ] Process/how-it-works section
- [ ] Before/after examples
- [ ] Pricing reference (link to pricing page)
- [ ] Social proof (testimonials, client count)
- [ ] FAQ section
- [ ] Schema: Service

## Content Tracking

### Monthly Review Metrics
| Metric | Tool | Target |
|--------|------|--------|
| Organic impressions | Google Search Console | +20% MoM |
| Organic clicks | Google Search Console | +25% MoM |
| Average position | Google Search Console | < 30 for target keywords |
| Pages indexed | Google Search Console | = published pages |
| Blog post traffic | GA4 | 100+ sessions within 30 days |
| Conversion rate (organic) | GA4 | > 2% signup rate |
| Keyword rankings (top 10) | Ahrefs/SEMrush | +5 new keywords/month |

### Quarterly Content Audit
- Identify underperforming pages (< 50 sessions/month after 90 days)
- Refresh content with new data, examples, internal links
- Consolidate thin pages if needed
- Update year references (2026 → 2027 when applicable)
- Check and fix broken internal/external links
